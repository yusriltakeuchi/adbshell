## ADBShell
ADBShell adalah sebuah tools sederhana untuk melakukan connecting ADB via WIFI. Hal ini berfungsi untuk melakukan debug pada mobile apps via wifi.

### Make Using
 - Python3

### How to use
```c#
python3 adbshell.py
```
